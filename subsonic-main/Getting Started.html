Installation

To install Supersonic, double-click the Supersonic Installer and follow the on-screen instructions. The Installer will tell you when the installation process is complete. Supersonic is installed as a Windows service, and is started automatically when the computer starts. Double-click the Supersonic icon in the Windows System Tray to launch the Supersonic control panel.

By default, Supersonic is accessible on http://localhost/, but the port number and URL can be changed from the control panel.

If you are upgrading an existing installation of Supersonic, your previous settings (stored in c:\supersonic) will be preserved.
